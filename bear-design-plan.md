# 裸熊 Agent 2.0 设计方案

## "咱们裸熊" 视觉风格分析

### 核心角色
- **大大 (Grizzly)**: 棕色 #85562F，热情活泼，领导者
- **胖达 (Panda)**: 黑白 #2F2B2B + #FFFFFF，害羞可爱，科技宅
- **白熊 (Ice Bear)**: 白色 #ABABAB + #FFFFFF，冷酷寡言，实力派

### 视觉风格特征
1. **圆润柔和的线条**：角色身体饱满圆润，没有尖锐棱角
2. **温暖的色调**：以棕色、米色、绿色为主，营造温馨感
3. **简洁的背景**：柔和的渐变色，自然场景（森林、草地、天空）
4. **糖果色画风**：明亮但不刺眼的配色
5. **治愈基调**：整体给人温暖、放松的感觉

### 新设计方案

#### 整体风格：从"深海生物发光"→"森林治愈系"
- **主色调**：温暖棕色 + 森林绿 + 天空蓝
- **背景**：从深海蓝黑 → 柔和的奶油白/浅米色
- **氛围**：从科技酷炫 → 温暖治愈

#### 配色方案
- **背景色**：#FFF8F0 (奶油白) / #F5EFE6 (暖米色)
- **主色**：#85562F (棕熊棕) 
- **辅助色**：#4A7C59 (森林绿)
- **强调色**：#F4A261 (暖橙)
- **文字色**：#2F2B2B (深棕黑)
- **卡片色**：#FFFFFF 带柔和阴影

#### 段位系统（用三只熊的成长阶段）
- 青铜 → 小熊崽（Baby Bear）
- 白银 → 幼熊（Young Bear）
- 黄金 → 成年熊（Adult Bear）
- 铂金 → 聪明熊（Smart Bear）
- 钻石 → 学霸熊（Scholar Bear）
- 星耀 → 大师熊（Master Bear）
- 王者 → 传奇熊（Legend Bear）

#### 皮肤系统
- 大大皮肤（Grizzly）：热情活泼型
- 胖达皮肤（Panda）：科技宅型
- 白熊皮肤（Ice Bear）：冷酷实力型

#### 情绪系统
- 开心：熊抱/举手欢呼
- 思考：托腮沉思
- 疲劳：打哈欠/趴下
- 升级：闪光特效/跳跃
