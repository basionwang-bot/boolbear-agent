## 项目背景
"龙虾 Agent"是一个教育+养成平台，学生通过与 AI 对话来学习，对话越多，小龙虾（电子宠物）越强。
核心关键词：**深海**、**成长**、**游戏化**、**学习陪伴**

## 方案一：「深海生物发光」—— Bioluminescence Design

### Design Movement
受深海生物发光现象（Bioluminescence）启发，融合赛博朋克与自然主义。整体氛围如同置身深海实验室，界面在黑暗中散发柔和的生物光芒。

### Core Principles
1. **暗色沉浸**：以深邃的海洋蓝黑为底，所有交互元素通过"发光"来引导注意力
2. **有机形态**：避免直角和机械感，使用流体形状和柔和的曲线模拟海洋生物
3. **层次光影**：通过多层半透明叠加和光晕效果创造深度感
4. **生命律动**：界面元素有呼吸般的微动效，暗示生命力和成长

### Color Philosophy
- 主背景：深海蓝黑 `#0a0e27` → `#0f1635`
- 生物光主色：青蓝色 `#00d4ff`（模拟水母发光）
- 强调色：珊瑚橙 `#ff6b35`（龙虾的颜色，用于关键 CTA）
- 辅助光：薄荷绿 `#00ffa3`（经验值、成长指标）
- 警告/能量：琥珀金 `#ffd700`（段位、成就）
- 情感逻辑：暗色代表深海的神秘与专注，发光色代表知识的启迪与成长

### Layout Paradigm
- 着陆页：全屏沉浸式深海场景，龙虾从深海中浮现
- 对话页：左侧为龙虾状态面板（垂直布局），右侧为对话流，底部为输入区
- 养成看板：卡片式仪表盘，每张卡片带有微光边框和玻璃拟态效果
- 广场页：瀑布流布局展示其他学生的龙虾

### Signature Elements
1. **发光边框（Glow Border）**：所有卡片和按钮带有柔和的外发光效果，hover 时光芒增强
2. **粒子气泡**：背景中缓慢上升的光点粒子，模拟深海气泡
3. **波纹涟漪**：点击交互产生水波纹扩散效果

### Interaction Philosophy
每一次交互都像是在深海中触发了一次生物发光反应。点击按钮时光芒闪烁，发送消息时气泡上升，龙虾升级时全屏光芒绽放。

### Animation
- 背景粒子：持续缓慢上升的发光粒子（CSS animation）
- 卡片入场：从下方淡入并微微上浮（framer-motion staggerChildren）
- 按钮 hover：外发光从 4px 扩展到 12px，颜色从青蓝渐变到珊瑚橙
- 龙虾呼吸：scale 在 1.0-1.03 之间循环（2s ease-in-out infinite）
- 经验条填充：流体动画从左到右填充，带有光泽扫过效果

### Typography System
- 标题字体：**Space Grotesk**（几何感强，科技+有机的平衡）
- 正文字体：**Noto Sans SC**（中文可读性极佳）
- 数据字体：**JetBrains Mono**（等级、经验值等数字数据）
- 层级：H1 48px/Bold → H2 32px/SemiBold → H3 24px/Medium → Body 16px/Regular

## 方案二：「像素复古街机」—— Pixel Arcade Design

### Design Movement
受 80-90 年代日式 RPG 和街机游戏启发，融合像素艺术与现代 UI 设计。整体氛围如同进入了一个复古但精致的游戏世界。

### Core Principles
1. **像素与现代的碰撞**：关键装饰元素使用像素风格，但 UI 交互保持现代流畅
2. **色彩饱和度高**：使用鲜明的游戏色板，每个功能区有独特的颜色标识
3. **信息密度适中**：模拟 RPG 状态栏的信息展示方式，但不过度拥挤
4. **成就驱动**：每个页面都有可见的"进度条"或"任务提示"

### Color Philosophy
- 主背景：暖灰白 `#f5f0e8`（羊皮纸质感）
- 主色调：宝石红 `#e63946`（活力、战斗感）
- 辅助色：皇家蓝 `#457b9d`（智慧、学习）
- 成长色：翡翠绿 `#2a9d8f`（经验、升级）
- 金色：`#e9c46a`（成就、奖励）
- 深色文字：`#264653`
- 情感逻辑：复古色调唤起童年游戏记忆，降低学习焦虑

### Layout Paradigm
- 着陆页：模拟游戏开始画面，"Press Start"风格的 CTA
- 对话页：模拟 RPG 对话框，底部弹出式文字，带有角色头像
- 养成看板：模拟角色状态面板，HP/MP 风格的属性条
- 广场页：模拟游戏大厅/排行榜界面

### Signature Elements
1. **像素边框装饰**：卡片四角使用像素风格的装饰角标
2. **RPG 状态条**：经验值、智慧值等使用游戏风格的彩色进度条
3. **8-bit 风格图标**：导航和功能图标采用像素化设计

### Interaction Philosophy
每一次操作都像是在游戏中完成了一个动作。发送消息有"确认音效"的视觉反馈，升级时有像素爆炸特效，完成任务时弹出"Quest Complete!"提示。

### Animation
- 文字出现：逐字打印效果（typewriter），模拟 RPG 对话
- 升级特效：像素星星从角色四周爆发
- 经验条：分段填充，每段有闪烁效果
- 页面切换：水平滑动，模拟横版游戏场景切换
- 按钮点击：微微下沉 2px + 阴影减小，模拟物理按压

### Typography System
- 标题字体：**Press Start 2P**（像素字体，用于标题和强调）
- 正文字体：**Noto Sans SC**（中文正文保持可读性）
- 数据字体：**VT323**（等级数字、计分板）
- 层级：H1 32px/Bold → H2 24px/Bold → Body 16px/Regular → Caption 12px

## 方案三：「海洋手绘绘本」—— Illustrated Ocean Storybook

### Design Movement
受儿童科普绘本和水彩插画启发，融合手绘质感与现代 Web 交互。整体氛围温暖、亲切，像翻开一本关于海洋冒险的故事书。

### Core Principles
1. **手绘温度**：用手绘风格的装饰元素（波浪线、涂鸦边框）替代冰冷的直线
2. **故事叙事感**：每个页面都像故事书的一页，有场景、角色和叙事
3. **柔和色调**：使用水彩般的柔和色彩，避免过于刺眼的高饱和度
4. **鼓励式设计**：每个交互反馈都是正向的、温暖的

### Color Philosophy
- 主背景：奶油白 `#fdf6ec`（纸张质感）
- 海洋蓝：`#5b9bd5`（天空与海面）
- 珊瑚粉：`#f4845f`（龙虾、温暖感）
- 海藻绿：`#7ec8a0`（成长、自然）
- 沙滩黄：`#f7d794`（成就、阳光）
- 深棕文字：`#5d4037`（手写笔迹感）
- 情感逻辑：绘本色调营造安全感和探索欲，适合教育场景

### Layout Paradigm
- 着陆页：全幅插画式 Hero，龙虾站在沙滩上望向大海，CTA 融入场景
- 对话页：笔记本/日记本风格，对话气泡有手绘边框
- 养成看板：贴纸/手账风格的属性展示，用星星和徽章代替数字
- 广场页：海滩场景，每只龙虾占据一个"沙滩位"

### Signature Elements
1. **手绘波浪分隔线**：页面区块之间用 SVG 手绘波浪线分隔
2. **贴纸徽章**：成就和段位用贴纸风格的徽章展示，带有微微倾斜的角度
3. **水彩渐变背景**：关键区域使用水彩晕染效果的渐变

### Interaction Philosophy
每一次交互都像是在故事书上贴了一张新贴纸。发送消息时气泡"贴"上去，升级时新贴纸飞入相册，完成任务时盖上一个"完成"印章。

### Animation
- 元素入场：从页面边缘"翻入"，模拟翻书效果
- 贴纸效果：新元素出现时有轻微的弹跳和旋转
- 波浪背景：底部波浪持续缓慢起伏（CSS animation）
- 龙虾动作：idle 状态下左右轻微摇摆，被点击时开心跳跃
- 印章效果：完成任务时印章从上方落下并微微弹跳

### Typography System
- 标题字体：**Caveat**（手写风格英文）+ **ZCOOL KuaiLe**（中文手写风格）
- 正文字体：**Noto Sans SC**（保持中文可读性）
- 强调字体：**Pacifico**（用于特殊标题和装饰文字）
- 层级：H1 40px/Bold → H2 28px/SemiBold → Body 16px/Regular → Note 14px/Light
